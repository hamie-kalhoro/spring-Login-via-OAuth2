Getting Started
Prerequisites
Java 8 or higher

Maven

GitHub account

Setup Instructions
Clone the repository:
<br>
git clone https://github.com/hamie-kalhoro/spring-Login-via-OAuth2.git
cd spring-Login-via-OAuth2
<br>
Build and run the application:
<br>
mvn spring-boot:run
